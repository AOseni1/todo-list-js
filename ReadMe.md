# To-do List (Vanilla JS)

![Screenshot](Screenshot2.png)

## Overview

A simple **Todo List** web app built with vanilla JavaScript, HTML, and CSS (Bootstrap).  
This project demonstrates **DOM manipulation, event handling, and basic JS logic**.

## [Live site](https://aoseni1.github.io/todo-list-js/)

## Features

- Add new todos dynamically
- Delete todos
- Filter/search todos in real-time
- Responsive layout for mobile and desktop
- Styled with Bootstrap and custom CSS

## Tech Stack

- HTML5
- CSS3 + Bootstrap 5
- JavaScript (ES6)

## Status

- Functional prototype
- Future improvements:
- Component-based structure
- Accessibility enhancements
- React + TypeScript refactor
- Persistent storage with `localStorage`

## How to Run

1. Clone the repo: `git clone https://github.com/USERNAME/todo-list-vanilla-js.git`
2. Open `index.html` in your browser

## Lessons Learned

- DOM selection and manipulation (`querySelector`, `innerHTML`)
- Handling events (`submit`, `click`, `keyup`)
- Dynamic class toggling for filtering
- Mobile-first layout considerations
